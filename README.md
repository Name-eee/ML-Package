### 粒子群优化
粒子群优化（PSO, Particle Swarm Optimization）算法是一种模拟鸟群觅食行为的优化算法，由Kennedy和Eberhart于1995年提出。它是一种群体智能算法，旨在通过群体中每个个体（粒子）的位置和速度更新规则，寻找问题的最优解。

- ### 线搜索法
  - 如果有梯度下降法的基础会很容易明白，剃度下降法就是线搜索方法中的最速下降法。
  - 线搜索方法的更新公式：
    $$ p_{n+1}=\alpha l + p_{n} $$ 
    其中：
    - $ p_n $为当前位置，$ p_{n+1} $为下一步位置即所求位置；
    - $ l $是搜索方向，可以由直接求梯度、牛顿法、拟牛顿法得到；
    - $ \alpha $ 为搜索步长，可以结合搜索方向用arjimo、curvature、wolfe等限制条件得到。
  - 搜索步长与搜索方向的函数已经在main.py中给出，应用在"__name__ == '__main__'"部分。
