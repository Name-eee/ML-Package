  - ### 线搜索法
    - 如果有梯度下降法的基础会很容易明白，梯度下降法就是线搜索方法中的最速下降法。
    - 线搜索方法的更新公式：
      $$p_{n+1}=\alpha l + p_{n}$$ 
      其中：
      - $p_n$为当前位置，$p_{n+1}$为下一步位置即所求位置；
      - $l$ 是搜索方向，可以由直接求梯度、牛顿法、拟牛顿法得到；
      - $\alpha$ 为搜索步长，可以结合搜索方向用arjimo、curvature、wolfe等限制条件得到。
    - 搜索步长与搜索方向的函数已经在main.py中给出，应用在"__name__ == '__main__'"部分。

## 2025-4-14
> 更新了NLP最大匹配分词法乱炖版-MMatching（配方：正向+逆向最大匹配，以及一点点的马尔可夫-bigram、FSM）

  - ### 最大匹配
  最基础的基于词典的分词算法。核心思想：尽可能匹配词典中最长的词。
  
   - 正向最大匹配 (FMM)从句首开始，取词典中最长词的长度 $L$，截取前 $L$ 个字。如果在词典中，则切分；如果不在，则减去最后一个字，继续匹配。其缺点为容易在句首遇到歧义时出错。
    
   - 逆向最大匹配 (RMM)从句尾开始向前匹配。统计显示，由于汉语的语言习惯，逆向匹配的错误率通常低于正向匹配。
    
   - <b>双向最大匹配</b>同时运行两者，如果结果一致则输出；如果不一致，通常选择切分词数更少、单字更少的那个。
    
  - ### Bigram (二元语法模型)
   - Bigram 是 N-gram 语言模型的一种特例（$N=2$）。它不再只依赖词典，而是考虑词与词之间的概率关系。
   - 数学表示：
     $$P(w_1, w_2, ..., w_n) \approx \prod_{i=1}^{n} P(w_i | w_{i-1})$$
   - 应用： 通过统计词组前后缀频率（大数定律：对于大样本频率趋于概率）来解决分词歧义。例如“结合成分子”，可以切分为“结合/成分/子”或“结合/成/分子”。
     
  - ### FSM (有限状态机)
   - FSM 是一种数学模型，用于表示有限个状态以及在这些状态之间的转移和动作。在 NLP 中， FSM 常用于构建高效的词典搜索或处理简单的形态学规则。
   - 此处主要用于解决对于英文专有名词及数字串（电话、日期等）这类特殊词组的分词。
